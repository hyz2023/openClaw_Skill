# 股息投资核心指标详解

## 一、基础指标

### 1. 股息率 (Dividend Yield)
```
股息率 = 年度每股股息 / 当前股价 × 100%
```

**解读**:
- **<2%**: 低股息，通常为成长股
- **2-4%**: 中等股息，平衡型
- **4-6%**: 高股息，收益型
- **>6%**: 超高股息，需警惕可持续性

**注意**: 过高股息率可能是"股息陷阱"(股价大跌导致)

### 2. 派息比率 (Payout Ratio)
```
派息比率 = 每股股息 / 每股收益 (EPS) × 100%
```

**解读**:
- **30-50%**: 理想区间，公司有足够资金再投资
- **50-70%**: 可接受，成熟公司常见
- **>70%**: 警戒线，可能影响未来增长
- **>100%**: 危险信号，分红超过盈利

**例外**: REITs 和公用事业公司通常派息比率较高 (70-90%)

### 3. 股息增长率 (Dividend Growth Rate)
```
增长率 = (今年股息 - 去年股息) / 去年股息 × 100%
```

**5 年平均增长率**更能反映趋势

**解读**:
- **≥10%**: 优秀，股息快速增长
- **5-10%**: 良好，稳定增长
- **3-5%**: 一般，勉强跑赢通胀
- **<3%**: 较差，购买力下降

## 二、稳定性指标

### 4. 连续分红年数 (Consecutive Years)
**美股分级**:
- **股息王 (Dividend Kings)**: 50+ 年连续增长
- **股息贵族 (Dividend Aristocrats)**: 25+ 年连续增长
- **股息挑战者 (Dividend Challengers)**: 10-24 年连续增长

**A 股参考**:
- **10+ 年**: 非常稳定 (如银行、公用事业)
- **5-10 年**: 较为稳定
- **<5 年**: 需观察

### 5. 股息波动率 (Dividend Volatility)
计算过去 5-10 年股息增长的标准差

**解读**:
- **低波动**: 稳定增长，可预测性强
- **高波动**: 受周期影响大，需谨慎

## 三、财务健康指标

### 6. 自由现金流覆盖率 (FCF Coverage)
```
FCF 覆盖率 = 自由现金流 / 总股息支出
```

**解读**:
- **>2.0**: 非常安全
- **1.5-2.0**: 安全
- **1.0-1.5**: 勉强覆盖
- **<1.0**: 危险，可能借钱分红

### 7. 负债率 (Debt-to-Equity)
```
负债率 = 总负债 / 股东权益
```

**解读**:
- **<0.5**: 低负债，财务稳健
- **0.5-1.0**: 中等，可接受
- **>1.0**: 高负债，风险增加
- **>2.0**: 警戒线

### 8. 利息覆盖率 (Interest Coverage)
```
利息覆盖率 = 息税前利润 (EBIT) / 利息支出
```

**解读**:
- **>5**: 安全
- **3-5**: 可接受
- **<3**: 风险较高

## 四、估值指标

### 9. 市盈率 (P/E Ratio)
```
P/E = 股价 / 每股收益
```

**股息股合理区间**:
- **公用事业**: 15-20x
- **消费必需品**: 20-25x
- **金融**: 10-15x
- **REITs**: 用 P/FFO 替代

### 10. 市盈增长率 (PEG Ratio)
```
PEG = P/E / 盈利增长率
```

**解读**:
- **<1**: 可能被低估
- **1-2**: 合理估值
- **>2**: 可能被高估

## 五、综合评分模型

### 股息确定性评分 (0-100)

| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 连续分红年数 | 25% | 50 年+=100, 25 年+=80, 10 年+=60, 5 年+=40 |
| 股息增长率 | 20% | 10%+=100, 7%+=80, 5%+=60, 3%+=40 |
| 派息比率 | 20% | 30-50%=100, 50-70%=70, 70-90%=40 |
| FCF 覆盖率 | 20% | >2=100, 1.5-2=80, 1-1.5=60, <1=30 |
| 负债率 | 15% | <0.5=100, 0.5-1=70, >1=40 |

**总分解读**:
- **90-100**: 极高确定性，核心持仓
- **75-89**: 高确定性，重点配置
- **60-74**: 中等确定性，适量配置
- **40-59**: 低确定性，谨慎观察
- **<40**: 避免

## 六、目标价位计算

### 方法 1: 股息率目标价
```
目标价 = 年度股息 / 目标股息率
```
例如：年度股息$2，目标股息率 4% → 目标价 = $2/0.04 = $50

### 方法 2: DCF 折现模型
```
内在价值 = Σ(未来股息 / (1+折现率)^年数) + 终值
```

### 方法 3: 历史股息率区间
- 查看过去 5 年平均股息率
- 当当前股息率 > 历史平均 + 1 个标准差时，可能是买入机会

## 七、行业特殊指标

### REITs
- **FFO (营运资金)**: 替代 EPS
- **AFFO (调整后 FFO)**: 更准确的现金流
- **P/FFO**: 替代 P/E

### 银行
- **不良贷款率**: <2% 为佳
- **资本充足率**: >10% 为佳
- **净息差**: 反映盈利能力

### 公用事业
- **监管资产基数 (RAB)**: 稳定收益基础
- **允许回报率**: 监管规定的收益率

## 八、A 股特殊考虑

1. **分红政策差异**: A 股公司分红意愿普遍低于美股
2. **除权除息**: A 股分红后股价会除权，需考虑填权行情
3. **税收政策**: 
   - 持有>1 年：免税
   - 1 月 -1 年：10%
   - <1 月：20%
4. **财报季节**: A 股分红集中在 4-7 月 (年报后)

## 九、常见陷阱

### 股息陷阱 (Dividend Trap)
特征:
- 股息率异常高 (>8-10%)
- 股价持续下跌
- 盈利下滑或亏损
- 派息比率>100%

### 一次性特别股息
- 不可持续
- 不应计入正常股息率

### 周期性行业
- 盈利波动大
- 分红不稳定
- 需在周期底部评估
