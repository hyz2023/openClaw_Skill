# 磁盘监控与清理脚本

## 📋 功能说明

### disk_monitor.sh - 磁盘监控
- 检查根分区使用率
- 超过 80% 时发出警告
- 显示占用空间最大的目录

### safe_cleanup.sh - 安全清理
只清理以下**安全**的临时文件：
- ✅ apt 包管理器缓存
- ✅ systemd 日志（保留最近 3 天）
- ✅ /tmp 目录中超过 7 天的文件
- ✅ ~/.cache 目录中超过 30 天的文件
- ✅ npm 缓存
- ✅ pip 缓存
- ✅ Docker 悬空镜像
- ✅ /var/log 中超过 30 天的日志
- ✅ 缩略图缓存

**不会删除：**
- ❌ 系统文件
- ❌ 配置文件
- ❌ 用户个人数据
- ❌ 应用程序代码

## 🚀 使用方法

### 手动检查磁盘
```bash
./disk_monitor.sh
```

### 手动清理
```bash
./safe_cleanup.sh
```

## ⏰ 自动监控

已设置每日自动监控（cron job），超过 80% 时会提醒你。

## 📊 当前状态

运行 `df -h` 查看实时磁盘使用情况。
